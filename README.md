3- Liste et Briève description des fonctionnalités IA 
# Correcteur Orthographique
- Utilisation de dictionnaires malgaches (scraping de tenymalagasy.org)
- Application de distance de Levenshtein pour les suggestions
- Détection automatique des fautes d'orthographe

# Vérification Phonotactique
- Détection des combinaisons interdites en malgache (ex: 'nb', 'mk', 'nk')
- Validation basée sur des règles et automates finis
- Suggestions de corrections phonétiques

# Lemmatisation
- Identification des racines des mots (ex: *manosika* → *tosika*)
- Reconnaissance des préfixes et suffixes courants
- Adaptation au *fandrasan-teny* malgache

# Autocomplétion
- Prédiction de mots suivants basée sur des modèles N-grams
- Entraînement sur corpus malgaches (Bible, journaux, lois)
- Suggestions contextuelles intelligentes

# Analyse de Sentiment
- Classification simple (Positif/Négatif/Neutre)
- Basée sur des listes de mots-clés (Bag of Words)
- Score de polarité et de confiance

# Explorateur Sémantique
- Graphe de connaissances basé sur des ontologies
- Suggestions de concepts liés (ex: "Razana" → "Famadinana")
- Navigation sémantique dans le texte
